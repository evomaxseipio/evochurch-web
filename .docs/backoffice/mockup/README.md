# BackOffice mockups

## Mockup unificado (referencia principal)

**[backoffice-mockup.html](./backoffice-mockup.html)** — Sales Hub + Organizations en un solo archivo interactivo.

Incluye:

| Módulo | Pantallas |
|--------|-----------|
| **Sales Hub** | Dashboard, Pipeline (kanban), Agenda |
| **Organizations** | List (datos), Empty, Loading, Error, Detail (tabs) |
| **Drawers** | Nueva / Editar organización, Registrar actividad |

Usa la barra inferior para saltar entre vistas. El sidebar navega las secciones principales.

## Archivos legacy

- `sales-hub-mockup.html` — solo Sales Hub (reemplazado por el unificado)
- `organizations-feature-mockup.html` — solo Organizations (reemplazado por el unificado)

Abre `backoffice-mockup.html` en el navegador para explorar el diseño completo.
